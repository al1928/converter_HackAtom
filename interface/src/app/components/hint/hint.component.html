Как выполнить конвертацию?

Перетащите файл в конвертер. MP3, WAV и аудио файлы будут преобразованы в файлы выбранного формата. По умолчанию в файл txt формата.

Почему можно выбрать файлы только MP3 и WAV форматов?

В данный момент сервис работает только с аудио файлами формата MP3 или WAV.
hourglass_empty

Сколько нужно ждать конвертации файла?

Процесс преобразование аудио файла в текст занимает времени пропорционально длительности аудио файла. Файл длительностью 5 минут будет конвертироваться примерно 4 минуты.

Можно ли задать последовательность конвертирумых файлов?

Захватите элемент с названием файла и перетащите его в нужную для вас позицию в списке файлов.
download

Как конвертировать только один файл из списка?

В строке файла справа есть соответствующая кнопка. После ее нажатия конвертирование будет производиться только для выбранного файла.
clear

Как можно удалить файл из списка?

Для удаления одного файла справа в строке файла есть соответсвуюшая кнопка. Для удаления всех файлов можно нажать на кнопку "очистить" или обновить страницу.
